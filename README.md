# CSGO Demo Cache

这是一个用于存储和管理CSGO比赛录像(demos)的仓库。

## 功能

- 存储CSGO比赛录像文件
- 提供录像文件的版本管理
- 方便团队协作和分享

## 使用说明

1. 将CSGO录像文件(.dem)添加到仓库中
2. 使用Git进行版本控制
3. 可以通过Git历史查看录像的变更

## 文件结构

```
├── README.md          # 项目说明
└── .gitignore         # Git忽略文件配置
```

## 注意事项

- 录像文件可能较大，请确保有足够的存储空间
- 定期清理不需要的旧录像以节省空间
